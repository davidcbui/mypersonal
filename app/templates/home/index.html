About Me

Welcome and I am David Bui, your host. This is my personal site and I think you may just be curious about me.
I am a professional software engineer and have been busy with many projects in medical health industry. I love LAMP stack and it has been my favorite technology for building software. I started my software engineering with C/C++ in the embedded firmware and have diversified my programming to web application in later years. So Java , C# , PHP , Python eventually added to my daily programming together with JAVASCRIPT, HTML, CSS, BOOTSTRAP.

I enjoy frontend tech stack as much as I do with backend. The latest frontend technologies I adapted to use on my development are ReActJS , VueJS , NodeJS and I make use of them depends on their suitability for project.
Projects that I have built are:

  • Beauty Salon Management (MEAN stack NodeJS)
  • Medical Record Retrieval Managements (LAMP stack - Laravel based)
  • Truckie Milages Auto Paysheet (LAMP stack - Symfony )
  • Appointment Bookings (LAMP stack - Laravel)
  • Misc Tool for DB Analysis (Flask - Python with Panda)
  • ...many more
